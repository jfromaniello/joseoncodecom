En esta ocasión quería invitarlos al CodeCamp que se realizará el próximo 15 de Octubre de 2011 en la Universidad Abierta Interamericana.

El CodeCamp es uno de los eventos de desarrollo de software más importantes en Argentina y se viene realizando desde hace ya 5 años. Esta impulsado principalmente por Microsoft, el Grupo de Usuarios de Microsoft de Argentina, la Universidad Abierta Interamericana y Digital Idea Solutions.

Cabe destacar que Tellago, la empresa para la cual estoy trabajando actualmente, forma parte de los patrocinadores de esta edición, entre otras empresas como SouthWorks, Telerik, Devexpress, Lagash, Redgate, Jetbrains y muchas otras más.

TDDeando Aplicaciones Javascript

Lo que hace doblemente especial a este evento para mi, es que una de las sesiones que propuse fue seleccionada!

Mi sesión trata sobre Test Driven Development en Javascript. Elegí este tema ya que TDD y unit testing es algo que me apasiona desde hace bastante tiempo, y Javascript es un lenguaje que cada día utilizo más; ya sea para aplicaciones en el navegador o código server side.

Extracto de la sesión

¿Cansado de escribir JavaScript a lo cowboy? Hoy en día las aplicaciones web requieren mucho más lógica del lado del cliente ya sea una simple animación, interacciones asíncronas, notificaciones desde la nube o almacenamiento local. Esto nos plantea un importante desafío como desarrolladores web.

En esta sesión voy a mostrar cómo escribir código JavaScript de calidad y con confianza sobre cada paso que damos. Además también mostraré como sortear ciertos obstáculos únicos que plantea escribir tests unitarios en javascript.
